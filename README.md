## Mono Repo for Flemme ecosystem

Flemme is a vanilla.js framework-agnostic form-management library.

To get started, head to [the core package](https://github.com/SacDeNoeuds/flemme/tree/main/modules/flemme).

Bindings are:

- [React](https://github.com/SacDeNoeuds/flemme/tree/main/modules/flemme-react)
- [Vue](https://github.com/SacDeNoeuds/flemme/tree/main/modules/flemme-vue)
- [HTML/DOM](https://github.com/SacDeNoeuds/flemme/tree/main/modules/flemme-dom)
- Svelte (coming soon)
